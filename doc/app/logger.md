# 日志接口

本logger接口是应用定制，并增加压缩重复日志项功能。


### log
> function logger:log(level, ...)

日志接口（内部接口）


### trace
> function logger:trace(...)

输出trace等级日志


### debug
> function logger:debug(...)

输出调试等级日志


### info
> function logger:info(...)

输出信息等级日志


### notice
> function logger:notice(...)

输出通知等级日志


### warning
> function logger:warning(...)

输出警告等级日志


### error
> function logger:error(...)

输出错误等级日志
